# T1 2024英雄联盟世界赛夺冠深度分析

这是一个基于Streamlit的数据可视化项目，深度分析T1战队在2024英雄联盟世界赛的夺冠表现。

## 功能特点
- 选手个人表现分析
- 团队协同数据可视化
- 英雄池与BP策略分析
- 多维度数据对比

## 技术栈
- Python
- Streamlit
- Plotly
- Pandas
